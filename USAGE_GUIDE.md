# RAGE MP Dumper - Usage Guide

## Installation Instructions

1. **Clone the Repository**:  
   Use the following command to clone the RAGE MP Dumper repository to your local machine:  
   ```bash
   git clone https://github.com/ghostmarcos11-dev/rage-mp-dumper.git
   ```  

2. **Navigate to the Project Directory**:  
   Change your directory to the cloned repository:  
   ```bash
   cd rage-mp-dumper
   ```  

3. **Install Dependencies**:  
   Make sure to install all the required dependencies. Typically, this can be done using a package manager such as npm or yarn (if applicable):  
   ```bash
   npm install
   ```
   or  
   ```bash
   yarn install
   ```  

4. **Building the Project**:  
   If there are any build steps required, make sure to run them:  
   ```bash
   npm run build
   ```
   or according to the specific build instructions provided in the repository.

5. **Starting the Application**:  
   Use the following command to start the RAGE MP Dumper:  
   ```bash
   npm start
   ```
